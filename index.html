Functions

Functions are one of the fundamental building blocks in JavaScript. A function is a JavaScript procedureâ€”a set of statements that performs a task or calculates a value. To use a function, you must define it somewhere in the scope from which you wish to call it.

Objects

Objects in JavaScript, just as in many other programming languages, can be compared to objects in real life. The concept of objects
in JavaScript can be understood with real life, tangible objects.

A JavaScript object has properties associated with it. A property of an object can be explained as a variable that is attached to the
object. Object properties are basically the same as ordinary JavaScript variables, except for the attachment to objects. The
properties of an object define the characteristics of the object. You access the properties of an object with a simple dot-notation.

Arrays

In JavaScript, array is a single variable that is used to store different elements. It is often used when we want to store list of elements
and access them by a single variable. Unlike most languages where array is a reference to the multiple variable, in JavaScript array
is a single variable that stores multiple elements.

For Loops

A for loop repeats until a specified condition evaluates to false. The JavaScript for loop is similar to the Java and C for loop.

Conditionals
